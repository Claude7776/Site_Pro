# 🌐 Sky Blue Corporation - Site de Présentation

Ce projet est un site web vitrine développé en HTML pour présenter **Sky Blue Corporation**, ses activités et ses services dans le domaine des technologies de l'information.

## 📖 Description

Le site a été conçu pour offrir une présentation claire et moderne de l'entreprise. Il met en avant :

- L'identité de Sky Blue Corporation
- Les services proposés
- Les compétences techniques
- Les informations de contact
- Une interface simple et responsive

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- React
- vite

## 🚀 Installation

Clonez le dépôt :

```bash
git clone https://github.com/Claude7776/Site_Pro.git
```

Accédez au dossier :

```bash
cd Site_Pro
```

Ouvrez ensuite lancé `npm run dev -- --port 5173 &
sleep 2 && curl -s http://localhost:5173 | head -5 dans votre terminal celà ouv.

## 🎯 Objectif

L'objectif de ce site est de fournir une présence en ligne professionnelle pour Sky Blue Corporation et de présenter ses activités de manière simple et efficace.
